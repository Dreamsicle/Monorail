## Main features
- DIY Stacks
- Small (3.8 MB with all modules installed, 11 KB alone)
- Fast (Starts in under a second, serves the included index.html in 3ms)
- NodeJS as a PHP replacement (soon)

## Installing
labHTTP is easy to get up and running.
1. Clone this repo
2. `npm install`
3. Run `node index.js`
